# Portfolio Site

Personal portfolio website for presenting my background, technical stack, and selected projects.

## Stack

- HTML
- CSS
- JavaScript

## Highlights

- Responsive navigation with mobile menu handling
- Scroll-based active section highlighting
- Reveal-on-scroll animations
- Featured project section linking to my main GitHub work

## Project Structure

- `index.html` main page markup
- `style/index.css` layout, typography, colors, and responsive styling
- `js/portfolio.js` navigation, reveal behavior, and footer year

## Running Locally

Open `index.html` in a browser.

## Notes

- The site is intentionally simple and static so it can be deployed easily on GitHub Pages or similar static hosts.
